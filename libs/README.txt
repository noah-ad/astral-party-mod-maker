此目录存放编译/运行所需的第三方库，仅为方便 clone 后直接编译而打包。
版权归各自作者所有，遵循其原始开源协议，请勿单独分发。

文件来源：
- AssetsTools.NET.dll          来自 AssetsTools.NET (https://github.com/nesrak1/AssetsTools.NET, MIT)
- AssetsTools.NET.Texture.dll  AssetsTools.NET 的纹理扩展 (同上)
- AssetRipper.TextureDecoder.dll  来自 AssetRipper (https://github.com/AssetRipper/TextureDecoder)
- SixLabors.ImageSharp.dll     来自 SixLabors.ImageSharp (https://github.com/SixLabors/ImageSharp)
- classdata.tpk                Unity 类型数据库 (随 UABEA 分发)

以上文件可从 UABEA (https://github.com/nesrak1/UABEA) 发布包中获取。
如需更新版本，替换同名文件即可（API 兼容前提下）。
